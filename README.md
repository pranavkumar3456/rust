# rust
rust robust system

Installing a Rust:

Windows:

https://www.rust-lang.org/tools/install

For Linux and Mac:

$ curl --proto '=https' --tlsv1.3 https://sh.rustup.rs -sSf | sh

Once Completed you can validate the installation by using following command i.e., 

$ rustc --version
rustc 1.79.0 (129f3b996 2024-06-10)
